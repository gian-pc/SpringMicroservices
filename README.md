# SpringMicroservices

Arquitectura de microservicios con Spring Boot 3, Maven multi-módulo, Eureka, Config Server, API Gateway y comunicación entre servicios.

## Estado actual
Proyecto base generado con Maven archetype y validado con `mvn clean install`.

## Origen del proyecto

El proyecto **SpringMicroservices** fue creado con Maven usando el siguiente comando:

```bash
mvn archetype:generate -DgroupId=com.mycompany.app -DartifactId=SpringMicroservices -DarchetypeArtifactId=maven-archetype-quickstart -DarchetypeVersion=1.5 -DinteractiveMode=false
```

## Objetivo
Construir una arquitectura de microservicios con Spring Boot 3 siguiendo una estructura multi-módulo con Maven.
